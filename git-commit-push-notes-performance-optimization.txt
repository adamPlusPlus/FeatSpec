Performance optimizations and code organization improvements

- Added VirtualList component for virtual scrolling to handle large lists efficiently
- Added DebouncedSave utility to batch rapid state changes and reduce localStorage writes
- Added EventListenerManager to track and properly cleanup event listeners, preventing memory leaks
- Added RenderQueue to batch render requests using requestAnimationFrame for better performance
- Updated managers (FileOperations, InitializationManager, ProjectManager, SectionManager, UIManager) with optimizations
- Updated UI components (ChatInput, ChatSettings, ChatTabBar, ChatWindow) with performance improvements
- Updated core modules (DataLayer, EventSystem, RenderingEngine) for better organization
- Updated AppConstants with new configuration values
- Added workers directory for background processing
- Updated index.html and server.js with related improvements
